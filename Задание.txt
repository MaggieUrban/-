Создайте 2 страницы

Первая страница:
1.1. Создайте заголовок 1-го уровня с фамилией и отчеством
1.2. Добавьте абзац с текстом "Начинающий Frontend-разработчик"
    Подчеркните слова "Frontend-разработчик"

1.3. Создайте навигацию с ненумерованным списком
        -Главная
        -Обо мне
        -Контакты
        -Галерея

1.4. Создайте раздел "Обо мне" 
        Добавьте заголовок 2-го уровня с текстом "Обо мне" 
        После заголовка разделительную горизонтальную линию hr
        Добавьте таблицу из 2-х столбцов 

        В 1м столбце напишите 3 строки с жирным текстом:
        Описание
        Навыки
        Описание проектов

        Во 2м столбце добавьте свой текстом

        После таблицы добавьте фотографию с описанием

1.4. Создайте раздел "Опыт" 
        Добавьте заголовок 2-го уровня с текстом "Опыт" 
        После заголовка разделительную горизонтальную линию hr

        Создайте нумеровованный список из 5ти элеметов, где каждым пунктом будет написано 
        "должность, место работы : период работы" 
        Пример: 
        Frontend-разработчик (студент), Urban University: Январь 2024г - по настоящее время

        Период работы выделите жирным шрифтом

1.5. Создайте раздел "Контакты" 
        Добавьте заголовок 2-го уровня с текстом "Контакты" 
        После заголовка разделительную горизонтальную линию hr

        Добавьте блок с номером телефона
        Пример: 
            <div>
                <span>Номер телефона: </span>
                <a href="tel:+1234567890"> +7 (123) 456 - 78 - 90 </a>
            </div>

       Используйте гиперссылку (тег <a>) с атрибутом href
       Для номера телефона в атрибуте href укажите tel:номер , это позволит при нажатии  открыть приложение для звонка на указанный телефонный номер.
     
        Аналогично добавьте блок с адресом электронной почты
        Для почты в атрибуте href укажите в mailto:почта,  при нажатии на гипертекст откроется приложение для отправки электронного письма
        Пример:    
        <div>
            <span>Email: </span>
            <a href="mailto:example@domain.com">example@domain.com</a>
        </div>


2.Создайте 2ю страницу 
2.1. Создайте заголовок 1-го уровня с фамилией и отчеством
2.2. Добавьте абзац с текстом "Начинающий Frontend-разработчик"
2.3. Добавьте навигацию из пункта 1.3
2.4. Добавьте 3 разных фотографии с описанием , задайте для них ширину и высоту (используйте атрибуты width и height для картинки)

3.1. Вернитесь к пункту 1.3
Добавьте для ссылки Главная атрибут href для перехода на 1ю страницу
Для ссылки "Обо мне" переход к разделу "Обо мне" 
Для ссылки "Контакты" переход к разделу "Контакты"
Для ссылки "Галерея" переход на 2ю страницу

Данная навигация должна работать как на первой, так и на второй странице
